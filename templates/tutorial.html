Tutorial

A blue background displays information about the data.



When loading data into the database from an entities.csv file, it is possible to add a synonym, description and unit column.
The synonym is displayed under the entity name. It is possible to search for an entity by name or synonym.



The description is visible as a tooltip when you hover the mouse over the entity.



The numeric and categorical tables in the database look like the figure below. They contain the columns: Patient_ID,Visit(optional),Date, Key and Value.



Data filtering


Filters are applied to each tab.
First, you can select a time window that shows only data from the selected range.

The numeric and categorical filters filter the data by Name_ID.

Next, if there is more than one visit. First, select which visit the filter will be applied to. Data can be filtered based on multiple visits.



Table browser

Selector Panel

Below is the selection panel for tab: Table browser. It contain a selector where you can choose which entities will be displayed in the table and what format the table should have: long format or entities as columns.


Long format: In this format, the table only contains columns Name_ID,Date,Visit(if exists), Key and Value.


Entities as columns: In this case entities are as columns.

Result also has a download button, by pressing this button, you can download the displayed table in the csv format.



Basic Stats
